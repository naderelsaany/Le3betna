# 🧠 ذاكرة جلسة التطوير (Session Memory)

تاريخ التحديث: يونيو 2026
المشروع: لعبتنا (Le3betna)

## 📌 ما تم إنجازه بنجاح (Completed)
- **البنية التحتية:** إعداد Flutter Web، ربط Firebase Realtime Database.
- **التصميم (Design System):** تم تطبيق معايير (Professional Dev Skills) الخاصة بـ Linear/Vercel (Dark Mode, Indigo Accent, Glassmorphism).
- **البروفايل:** برمجة شاشة `ProfileSettingsDialog` تدعم تعديل الاسم ورفع الصورة مع الضغط التلقائي للصورة (`imageQuality: 50`) وتحويلها لـ Base64 لعدم استهلاك مساحة التخزين.
- **محرك الألعاب (Game Engine):**
  - **الدومينو:** محرك كامل يدعم الأوراق، السحب، التعادل، القفلة، وتحديد الفائز بالنقاط الأقل، مع تصميم حقيقي للأوراق عبر `CustomPainter`.
  - **٤ في صف (Connect 4):** شبكة 6x7 مع خوارزمية ذكية لاكتشاف الفوز بكل الاتجاهات.

## ⏳ المتبقي حسب الخطة (Remaining Tasks)
1. **لعبة ليدو (Ludo):** تحتاج لبناء محرك مسارات، النرد، وتحديد مناطق الأمان. (هذه أضخم لعبة).
2. **التفاعلات الحية (Transient):** إتاحة إرسال إيموجي أو تفاعلات (زي رمي الشبشب أو الطماطم) تظهر على الشاشة كـ Animation مؤقت.
3. **الصوتيات (SoundManager):** إضافة مؤثرات صوتية (صوت سقوط القرص، صوت خبط الدومينو، موسيقى فوز/خسارة) مع كتم الصوت افتراضياً.
4. **صفحة الهبوط (Landing Page):** برمجة موقع Next.js منفصل لرفع المشروع عليه (Vercel) لجلب الزيارات وتحويلهم للعبة.
5. **كتابة Unit Tests:** لاختبار قوانين اللعب.
